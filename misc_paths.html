Остальные пути¶

GET /api​/v4​/join​/{token}​/preview¶

Информация приглашения.

{
  "ok": true,
  "result": {
    "icons": {
      "blurhash": "string",
      "color": "string",
      "letters": "string",
      "lg": {
        "height": 0,
        "url": "string",
        "width": 0
      },
      "sm": {
        "height": 0,
        "url": "string",
        "width": 0
      },
      "stub": "string"
    },
    "name": "string",
    "uid": "string"
  }
}
Параметры

  • token – Токен предварительного просмотра.

Ответный JSON объект

  • ok (boolean) – True если не было ошибки

  • result (object) – Информация приглашения.

GET ​/api​/v4​/ping¶

Пинг запрос на сервер

Пример ответа

HTTP/1.1 200 OK
Content-Type: text/javascript

{
  "ok": true,
  "result": "pong"
}
Ответный JSON объект

  • ok (boolean) – True если не было ошибки

  • result (string) – Задано на «pong»

Код статуса

  • 200 OK – Без ошибки
