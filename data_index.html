ButtonColors¶

Цвета кнопок для приложения

Поля:

  • brand_static (RGBColor) - статический цвет

  • brand_active (RGBColor) - активный цвет

  • brand_disable (RGBColor) - отключенный цвет

  • simple_static (RGBColor) - простой статический цвет

  • simple_active (RGBColor) - простой активный цвет

  • simple_disable (RGBColor) - простой отключенный цвет

CallDevice¶

Устройство участника звонка

Поля:

  • muted (boolean) - устройство заглушено

  • useragent (string) - Описание устройства

CallEvent¶

Информация аудио звонка

Поля:

  • start (string) - Начало звонка. Возможен null

  • finish (string) - Окончание звонка. Возможен null

  • audiorecord (boolean) - Запись звонка включена

  • onliners (CallOnliner) - Участники звонка. Возможно поле будет отсутствовать.

CallOnliner¶

Участник звонка

Поля:

  • jid (JID) - Id контакта

  • display_name (string) - Имя контакта

  • role (string) - Роль контакта

  • icon (string) - Иконка контакта

  • muted (boolean) - Микрофон выключен. Вычисляется из состояния устройств

  • devices (CallDevice) - Устройства участвующие, строго одно пока-что

Chat¶

Образ чата. (прямого, группового или задачного)

Поля:

  • jid (JID) - Id группы/задачи/контакта

  • chat_type (ChatType) - Тип чата

  • base_gentime (number) - Базовые поля (не относящиеся к каким либо участникам) версия. Возможно поле будет отсутствовать.

  • gentime (number) - Поля чата относящиеся к какому либо конкретному участнику) версия

  • created (string) - Дата создания, iso datetime

  • display_name (string) - Титул

  • icons (IconData) - Информация об иконках. Возможен null

  • counters_enabled (boolean) - Непрочитанные сообщения будут учитываться в счетчике. Возможно поле будет отсутствовать

  • can_call (boolean) - Могу ли я звонить в этот чат. Возможно поле будет отсутствовать

  • can_send_message (boolean) - Могу ли я отправлять сообщения в этот чат. Возможно поле будет отсутствовать

  • cant_send_message_reason (string) - Почему я не могу отправлять сообщения в этот чат (если не могу). Возможно поле будет отсутствовать

  • collapsed (boolean) - Сжатое описание. Используется только в задачах. Возможно поле будет отсутствовать

  • draft (string) - Черновик последнего сообщения, если сушествует. Возможно поле будет отсутствовать

  • draft_num (number) - Версия последнего черновика , если существует. Возможно поле будет отсутствовать

  • hidden (boolean) - Невидимый чат. Возможно поле будет отсутствовать

  • notifications_enabled (boolean) - Пуш уведомления включены. Возможно поле будет отсутствовать

  • num_importants (number) - Число важных сообщений. Возможно поле будет отсутствовать

  • num_unread (number) - Счетчик непрочитаных сообщений. Возможно поле будет отсутствовать

  • num_unread_notices (number) - Счетчик упоминаний (@). Возможно поле будет отсутствовать

  • last_message (Message) - Обьект последнего сообщения. Возможно поле будет отсутствовать. Возможен null

  • last_read_message_id (string) - Id последнего прочитанного сообщения, если существует. Возможно поле будет отсутствовать

  • section (string) - Id проекта / секции , если существует. Возможно поле будет отсутствовать

  • changeable_fields (string) - Список редактируемых полей. Возможно поле будет отсутствовать

  • pinned (boolean) - Закреплен ли чат на верху. Возможно поле будет отсутствовать

  • pinned_sort_ordering (number) - Тип сортирования прикрепленного чата. Возможно поле будет отсутствовать

  • num_members (number) - Количество не архивированных участников. Возможно поле будет отсутствовать. Возможен null

  • can_delete (boolean) - Могу ли я удалить этот чат. Возможно поле будет отсутствовать

  • description (string) - Описание группы или задачи. Возможно поле будет отсутствовать

  • markup (MarkupEntity) - Markup объекты для описания. Экспериментально. Возможно поле будет отсутствовать

  • feed (boolean) - Существует в feed (главный экран). Возможно поле будет отсутствовать

  • pinned_message (Message) - Прикрепленные сообщения этого чата. Возможно поле будет отсутствовать. Возможен null

  • color_index (number) - индекс нестандартного цвета из таблицы цветов. Только в задачах. Возможно поле будет отсутствовать. Возможен null

  • num_items (number) - количество пунктов в чеклисте. Только в задачах. Возможно поле будет отсутствовать. Возможен null

  • num_checked_items (number) - Количество отмеченных пунктов в чеклисте. Только в задачах. Возможно поле будет отсутствовать. Возможен null

  • assignee (JID) - Id контакта уполномоченного. Только в задачах. Возможно поле будет отсутствовать

  • num (number) - Номер задачи в этой команде. Возможно поле будет отсутствовать

  • observers (JID) - Id подписчиков.. Возможно поле будет отсутствовать

  • owner (JID) - Создатель задачи. Возможно поле будет отсутствовать

  • task_status (string) - Статус задачи. Возможно не стандартное. Возможно поле будет отсутствовать

  • title (string) - Титул задачи. Создается из номера и описания. Возможно поле будет отсутствовать

  • done (string) - Дата завершения задачи в iso формате, если существует. Возможно поле будет отсутствовать

  • done_reason (string) - Причина завершения задачи, если существует. Возможно поле будет отсутствовать

  • deadline (string) - Дедлайн задачи в iso формате, если существует. Возможно поле будет отсутствовать

  • deadline_expired (boolean) - Просрочен ли дедлайн. Возможно поле будет отсутствовать

  • links (MessageLinks) - Ссылки в описании. Возможно поле будет отсутствовать

  • tags (string) - Список тэгов задачи, если существуют. Возможно поле будет отсутствовать

  • importance (number) - Важность задачи, если доступно в команде. Возможно поле будет отсутствовать. Возможен null

  • urgency (number) - Срочность задачи, если доступно в команде. Возможно поле будет отсутствовать. Возможен null

  • spent_time (number) - Время затраченное на задачу. Возможно поле будет отсутствовать. Возможен null

  • complexity (number) - Сложность задачи. Возможно поле будет отсутствовать. Возможен null

  • linked_messages (any) - Используется для «Create task from messages…». Возможно поле будет отсутствовать

  • uploads (Upload) - UID загрузок для запроса, объекты загрузок для ответа. Возможно поле будет отсутствовать

  • items (TaskItem) - предметы чеклиста. Только в задачах. Возможно поле будет отсутствовать

  • parents (Subtask) - Родительские задачи. Возможно поле будет отсутствовать

  • tabs (TaskTabKey) - Имена закладок. Возможно поле будет отсутствовать

  • status (GroupStatus) - Мой статус в групповом чате. Возможно поле будет отсутствовать. Возможен null

  • members (GroupMembership) - Участники группового чата. Возможно поле будет отсутствовать

  • can_add_member (boolean) - Могу ли я добавлять участников в этот групповой чат. Возможно поле будет отсутствовать

  • can_remove_member (boolean) - Могу ли я убирать участников из этого группового чата. Возможно поле будет отсутствовать

  • can_change_member_status (boolean) - Могу ли я менять статус участников этого группового чата. Возможно поле будет отсутствовать

  • can_change_settings (boolean) - устарело: используете changeable fields. Возможно поле будет отсутствовать

  • default_for_all (boolean) - Будут ли новые участники добавлены в этот чат при создании. Возможно поле будет отсутствовать

  • readonly_for_members (boolean) - Только для чтения для не администраторов (как каналы в Telegram но переключаемые). Возможно поле будет отсутствовать

  • autocleanup_age (number) - Удалять сообщения через столько то секунд. Экспериментальная функция. Возможно поле будет отсутствовать. Возможен null

  • public (boolean) - Могут ли другие участники команды видеть этот чат/задачу. Возможно поле будет отсутствовать

  • can_join (boolean) - Могу ли я присоединиться к этому публичному чату/задаче. Возможно поле будет отсутствовать

  • can_delete_any_message (boolean) - Могу ли я удалять сообщения в этом чате. Возможно поле будет отсутствовать

  • can_set_important_any_message (boolean) - Могу ли я менять флаг важности сообщений в чате этом. Возможно поле будет отсутствовать

  • last_activity (string) - Дата последнего сообщения в этом чате если оно было удалено. Возможно поле будет отсутствовать

  • color_index (number) - Color index

  • section_enabled (boolean) - Project filter enabled. Maybe omitted. Might be null

  • section (string) - Project id if project filter enabled. Maybe omitted

  • tags_enabled (boolean) - Tags filter enabled. Maybe omitted. Might be null

  • tags (string) - Tag ids if tags filter enabled. Maybe omitted

  • task_status (string) - Task status. Maybe omitted

  • task_importance_enabled (boolean) - Task importance filter enabled. Maybe omitted. Might be null

  • task_importance (number) - Task importance if task importance filter enabled. Maybe omitted. Might be null

  • task_urgency_enabled (boolean) - Task urgency filter enabled. Maybe omitted. Might be null

  • task_urgency (number) - Task urgency if task urgency filter enabled. Maybe omitted. Might be null

Contact¶

Contact

Поля:

  • jid (JID) - Contact Id

  • display_name (string) - Full name in chats

  • short_name (string) - Short name in chats

  • contact_email (string) - Contact email in this team

  • contact_phone (string) - Contact phone in this team

  • icons (IconData) - Icons data. Might be null

  • role (string) - Role in this team

  • mood (string) - Mood in this team. Maybe omitted

  • status (TeamStatus) - Status in this team

  • last_activity (string) - Last activity in this team (iso datetime). Maybe omitted

  • add_to_team_rights (boolean) - Can contact add users to this team. Maybe omitted

  • is_archive (boolean) - Contact deleted. Maybe omitted

  • botname (string) - Bot name. Empty for users. Maybe omitted

  • sections (string) - Section ids

  • can_send_message (boolean) - Can I send message to this contact. Maybe omitted

  • cant_send_message_reason (string) - Почему я не могу отправлять сообщения в этот чат (если не могу). Возможно поле будет отсутствовать

  • can_call (boolean) - Can I call to this contact. Maybe omitted

  • can_create_task (boolean) - Can I call create task for this contact. Maybe omitted

  • can_add_to_group (boolean) - Can I add this contact to group chats. Maybe omitted

  • can_delete (boolean) - Can I remove this contact from team. Maybe omitted

  • changeable_fields (string) - Changeable fields. Maybe omitted

  • family_name (string) - Family name. Maybe omitted

  • given_name (string) - Given name. Maybe omitted

  • patronymic (string) - Patronymic, if any. Maybe omitted

  • default_lang (string) - Default language code. Maybe omitted. Might be null

  • debug_show_activity (boolean) - Enable debug messages in UI. Maybe omitted. Might be null

  • dropall_enabled (boolean) - Enable remove all messages experimental features. Maybe omitted. Might be null

  • alt_send (boolean) - Use Ctrl/Cmd + Enter instead Enter. Maybe omitted. Might be null

  • asterisk_mention (boolean) - Use * as @ for mentions. Maybe omitted. Might be null

  • always_send_pushes (boolean) - Send push notifications even contact is online. Maybe omitted. Might be null

  • timezone (string) - Timezone, if any. Maybe omitted. Might be null

  • quiet_time_start (string) - Quiet time start. Maybe omitted. Might be null

  • quiet_time_finish (string) - Quiet time finish. Maybe omitted. Might be null

  • group_notifications_enabled (boolean) - Push notifications for group chats. Maybe omitted. Might be null

  • task_notifications_enabled (boolean) - Push notifications for task chats. Maybe omitted. Might be null

  • contact_short_view (boolean) - Short view in contact list. Maybe omitted. Might be null

  • group_short_view (boolean) - Short view in group list. Maybe omitted. Might be null

  • task_short_view (boolean) - Short view in task list. Maybe omitted. Might be null

  • contact_mshort_view (boolean) - Short view in contact list in mobile app. Maybe omitted. Might be null

  • group_mshort_view (boolean) - Short view in group list in mobile app. Maybe omitted. Might be null

  • auth_2fa_enabled (boolean) - Two-factor authentication is configured and confirmed. Maybe omitted

  • auth_2fa_status (string) - Two-factor authentication status. Maybe omitted

  • task_mshort_view (boolean) - Short view in task list in mobile app. Maybe omitted. Might be null

  • contact_show_archived (boolean) - Show archived contacts in contact list. Maybe omitted. Might be null

  • unread_first (boolean) - Show unread chats first in feed. Maybe omitted. Might be null

  • munread_first (boolean) - Show unread chats first in feed in mobile app. Maybe omitted. Might be null

  • can_add_to_team (boolean) - Can I add new members to this team. Maybe omitted

  • can_manage_sections (boolean) - Can I manage sections in this team. Maybe omitted

  • can_manage_tags (boolean) - Can I manage tags in this team. Maybe omitted

  • can_manage_integrations (boolean) - Can I manage integrations in this team. Maybe omitted

  • can_manage_color_rules (boolean) - Can I manage color rules in this team. Maybe omitted

  • can_create_group (boolean) - Can I create group chats in this team. Maybe omitted

  • can_join_public_groups (boolean) - Can I view/join public group in this team. Maybe omitted

  • can_join_public_tasks (boolean) - Can I view/join public tasks in this team. Maybe omitted

  • can_delete_any_message (boolean) - Deprecated: use CanDeleteAnyMessage in chat object. Maybe omitted

  • max_mood_length (number) - Maximum chars for mood in team

  • max_message_length (number) - Maximum chars for text message

  • max_section_length (number) - Maximum length for project and contact’s sections names

  • max_tag_length (number) - Maximum length for tags

  • max_task_title_length (number) - Maximum length for task title

  • max_color_rule_description_length (number) - Maximum length for ColorRule description

  • max_url_length (number) - Maximum length for urls

  • max_integration_comment_length (number) - Maximum length for Integration comment

  • max_teams (number) - Maximum teams for one account

  • max_message_search_limit (number) - Maximum search result

  • afk_age (number) - Max inactivity seconds

  • auth_by_password (boolean) - Password authentication enabled. Maybe omitted

  • auth_by_qr_code (boolean) - QR-code / link authentication enabled. Maybe omitted

  • auth_by_sms (boolean) - SMS authentication enabled. Maybe omitted

  • auth_2fa (boolean) - Two-factor authentication (2FA) enabled. Maybe omitted

  • custom_server (boolean) - True for premise installation

  • installation_type (string) - Name of installation

  • installation_title (string) - Installation title, used on login screen. Maybe omitted

  • background (string) - Background image url, if any. Maybe omitted

  • firebase_app_id (string) - Firebase settings for web-push notifications

  • firebase_sender_id (string) - Firebase settings for web-push notifications

  • firebase_api_key (string) - Firebase settings for web-push notifications

  • firebase_auth_domain (string) - Firebase settings for web-push notifications

  • calls (boolean) - Calls functions enabled

  • mobile_calls (boolean) - Calls functions enabled for mobile applications

  • calls_record (boolean) - Calls record enabled

  • only_one_device_per_call (boolean) - Disallow call from multiply devices. Experimental. Maybe omitted

  • max_participants_per_call (number) - Maximum number of participants per call. Maybe omitted
