Псевдонимы типов¶

ActiveUserDailyStatList¶

Изначальный тип: ActiveUserDailyStat

Является списком

ISODateTimeString¶

Дата и время в RFC3339 формате. Пример: 2019-09-18T00:00:07.435409Z

Базовый тип: string

JID¶

Базовый тип: string

MessageLinks¶

Базовый тип: MessageLink

Является списком

PushDeviceType¶

Базовый тип: int

RGBColor¶

Цвет в #rrggbb формате где rr, gg, bb шестнадцатеричные цифры от 00 до ff красного, зеленого и синего канала соответственно. (например желтый будет #ffff00)

Базовый тип: string

TaskFilterKey¶

Базовый тип: string

TaskSortKey¶

Базовый тип: string

TaskTabKey¶

Базовый тип: string

Индекс JSON объектов¶

ButtonColors¶

Цвета кнопок для приложения

Поля:

  • brand_static (RGBColor) - статический цвет

  • brand_active (RGBColor) - активный цвет

  • brand_disable (RGBColor) - отключенный цвет

  • simple_static (RGBColor) - простой статический цвет

  • simple_active (RGBColor) - простой активный цвет

  • simple_disable (RGBColor) - простой отключенный цвет

CallDevice¶

Устройство участника звонка

Поля:

  • muted (boolean) - устройство заглушено

  • useragent (string) - Описание устройства

CallOnliner¶

Участник звонка

Поля:

  • jid (JID) - Id контакта

  • display_name (string) - Имя контакта

  • role (string) - Роль контакта

  • icon (string) - Иконка контакта

  • muted (boolean) - Микрофон выключен. Вычисляется из состояния устройств

  • devices (array[CallDevice]) - Устройства участвующие, строго одно пока-что

Chat¶

Образ чата. (прямого, группового или задачного)

Поля:

  • jid (JID) - Id группы/задачи/контакта

  • chat_type (ChatType) - Тип чата

  • base_gentime (number) 💥 - Базовые поля (не относящиеся к каким либо участникам) версия

  • gentime (number) - Поля чата относящиеся к какому либо конкретному участнику) версия

  • created (string) - Дата создания, iso datetime

  • display_name (string) - Титул

  • icons (IconData) 0️⃣ - Данные иконки

  • counters_enabled (boolean) 💥 - Непрочитанные сообщения будут учитываться в счетчике

  • can_call (boolean) 💥 - Могу ли я звонить в этот чат

  • can_send_message (boolean) 💥 - Могу ли я отправлять сообщения в этот чат

  • cant_send_message_reason (string) 💥 - Почему я не могу отправлять сообщения в этот чат (если не могу)

  • collapsed (boolean) 💥 - Сжатое описание. Используется только в задачах

  • draft (string) 💥 - Черновик последнего сообщения, если существует

  • draft_gentime (number) 💥 - Версия последнего черновика , если существует

  • hidden (boolean) 💥 - Невидимый чат

  • notifications_enabled (boolean) 💥 - Пуш уведомления включены

  • num_importants (number) 💥 - Число важных сообщений

  • num_unread (number) 💥 - Счетчик непрочитаных сообщений

  • num_unread_notices (number) 💥 - Счетчик упоминаний (@)

  • last_message (Message) 💥 - Объект последнего сообщения

  • last_read_message_id (string) 💥 - Id последнего прочитанного сообщения, если существует

  • section (string) 💥 - Id проекта / секции , если существует

  • changeable_fields (array[string]) 💥 - Список редактируемых полей

  • pinned (boolean) 💥 - Закреплен ли чат на верху

  • pinned_sort_ordering (number) 💥 - Тип сортирования прикрепленного чата

  • num_members (number) 💥 - Количество не архивированных участников

  • can_delete (boolean) 💥 - Могу ли я удалить этот чат

  • description (string) 💥 - Описание группы или задачи

  • markup (array[MarkupEntity]) 💥 - Markup объекты для описания. Экспериментально

  • feed (boolean) 💥 - Существует в feed (главный экран)

  • pinned_message (Message) 💥 - Прикрепленные сообщения этого чата

  • color_index (number) 💥 - индекс нестандартного цвета из таблицы цветов. Только в задачах

  • num_items (number) 💥 - количество пунктов в чеклисте. Только в задачах

  • num_checked_items (number) 💥 - Количество отмеченных пунктов в чеклисте. Только в задачах

  • assignee (JID) 💥 - Id доверенного контакта. Только в задачах

  • num (number) 💥 - Номер задачи в этой команде

  • observers (array[JID]) 💥 - Id подписчиков

  • owner (JID) 💥 - Создатель задачи

  • task_status (string) 💥 - Статус задачи. Возможно не стандартное

  • title (string) 💥 - Титул задачи. Создается из номера и описания

  • done (string) 💥 - Дата завершения задачи в iso формате, если существует

  • done_reason (string) 💥 - Причина завершения задачи, если существует

  • deadline (string) 💥 - Дедлайн задачи в iso формате, если существует

  • deadline_expired (boolean) 💥 - Просрочен ли дедлайн

  • links (MessageLinks) 💥 - Ссылки в описании

  • tags (array[string]) 💥 - Список тэгов задачи, если существуют

  • importance (number) 💥 - Важность задачи, если доступно в команде

  • urgency (number) 💥 - Срочность задачи, если доступно в команде

  • spent_time (number) 💥 - Время затраченное на задачу

  • complexity (number) 💥 - Сложность задачи

  • linked_messages (array[any]) 💥 - Используется для «Create task from messages…»

  • uploads (array[Upload]) 💥 - UID загрузок для запроса, объекты загрузок для ответа

  • items (array[TaskItem]) 💥 - предметы чеклиста. Только в задачах

  • parents (array[Subtask]) 💥 - Родительские задачи

  • tabs (array[TaskTabKey]) 💥 - Имена закладок

  • status (GroupStatus) 💥 - Мой статус в группе

  • members (array[GroupMembership]) 💥 - Участники группового чата

  • can_add_member (boolean) 💥 - Могу ли я добавлять участников в этот групповой чат

  • can_remove_member (boolean) 💥 - Могу ли я убирать участников из этого группового чата

  • can_change_member_status (boolean) 💥 - Могу ли я менять статус участников этого группового чата

  • can_change_settings (boolean) 💥 - устарело: используете changeable fields

  • default_for_all (boolean) 💥 - Будут ли новые участники добавлены в этот чат при создании

  • readonly_for_members (boolean) 💥 - Только для чтения для не администраторов (как каналы в Telegram но переключаемые)

  • autocleanup_age (number) 💥 - Удалять сообщения через столько то секунд. Экспериментальная функция

  • public (boolean) 💥 - Могут ли другие участники команды видеть этот чат/задачу

  • can_join (boolean) 💥 - Могу ли я присоединиться к этому публичному чату/задаче

  • can_delete_any_message (boolean) 💥 - Могу ли я удалять сообщения в этом чате

  • can_set_important_any_message (boolean) 💥 - Могу ли я менять флаг важности сообщений в этом чате

  • last_activity (string) 💥 - Дата последнего сообщения в этом чате если оно было удалено

  • draft_num (number) 💥 - Устарело

ColorRule¶

Набор правил о применении цветов к задачам

Поля:

  • uid (string) - Id правила

  • priority (number) - Приоритет правила

  • description (string) 💥 - Описание правила

  • color_index (number) - Индекс цвета

  • section_enabled (boolean) 💥 - Фильтр проектов включен

  • section (string) 💥 - Id проекта если фильтр по проектам включен

  • tags_enabled (boolean) 💥 - Фильтр по тэгам включен

  • tags (array[string]) 💥 - тэги если фильтр по тэгам включен

  • task_status (string) 💥 - Статус задачи

  • task_importance_enabled (boolean) 💥 - Фильтр по важности включен

  • task_importance (number) 💥 - Важность задачи если фильтр по важности включен

  • task_urgency_enabled (boolean) 💥 - Фильтр по срочности включен

  • task_urgency (number) 💥 - Срочность задачи если фильтр по срочности включен

Contact¶

Contact

Поля:

  • jid (JID) - Id контакта

  • display_name (string) - Полное имя в чатах

  • short_name (string) - Короткое имя в чатах

  • contact_email (string) - Контактный email в этой команде

  • contact_phone (string) - Контактный телефон в этой команде

  • icons (IconData) 0️⃣ - Данные иконки

  • role (string) - Роль этой команде

  • mood (string) 💥 - Настроение в этой команде

  • status (TeamStatus) - Статус в этой комманде

  • last_activity (string) 💥 - Последняя активность в этой команде (iso дата-время)

  • is_archive (boolean) 💥 - Пользователь был удален

  • botname (string) 💥 - Имя бота. Пустое для пользователей

  • sections (array[string]) - Id секции

  • can_send_message (boolean) 💥 - Могу ли я отправлять сообщения этому контакту

  • cant_send_message_reason (string) 💥 - Почему я не могу отправлять сообщения в этот чат (если не могу)

  • can_call (boolean) 💥 - Могу ли я звонить этому контакту

  • can_create_task (boolean) 💥 - Могу ли я создавать задачи этому контакту

  • can_import_tasks (boolean) 💥 - Могу ли я импортировать задачи в этой команде

  • can_add_to_group (boolean) 💥 - Могу ли я добавлять этого контакта в групповые чаты

  • can_delete (boolean) 💥 - Могу ли я убрать этот контакт из команды

  • changeable_fields (array[string]) 💥 - Изменяемые поля

  • family_name (string) 💥 - Фамилия

  • given_name (string) 💥 - Имя

  • patronymic (string) 💥 - Отчество если существует

  • default_lang (string) 💥 - Код языка по умолчанию

  • debug_show_activity (boolean) 💥 - Включить отладочные сообщения в интерфейсе

  • dropall_enabled (boolean) 💥 - Включить экспериментальную функцию убрать все сообщения

  • alt_send (boolean) 💥 - Использовать Ctrl/Cmd + Enter вместо Enter

  • asterisk_mention (boolean) 💥 - Использовать * вместо @ для упоминаний

  • always_send_pushes (boolean) 💥 - Посылать пуш уведомления даже если контакт онлайн

  • timezone (string) 💥 - Часовой пояс

  • quiet_time_start (string) 💥 - Начало тихого времени

  • quiet_time_finish (string) 💥 - Окончание тихого времени

  • group_notifications_enabled (boolean) 💥 - Пуш уведомления для групповых чатов

  • task_notifications_enabled (boolean) 💥 - Пуш уведомления для задач

  • contact_short_view (boolean) 💥 - Короткий список контактов

  • group_short_view (boolean) 💥 - Короткий список групп

  • task_short_view (boolean) 💥 - Короткий список задач

  • contact_mshort_view (boolean) 💥 - Короткий список контактов в мобильном приложении

  • group_mshort_view (boolean) 💥 - Короткий список групп в мобильном приложении

  • auth_2fa_enabled (boolean) 💥 - Двух факторная аутентификация настроена и подтверждена

  • auth_2fa_status (string) 💥 - Статус двух факторной аутентификации

  • task_mshort_view (boolean) 💥 - Короткий список задач в мобильном приложении

  • contact_show_archived (boolean) 💥 - Показывать архивированных контактов в списке контактов

  • unread_first (boolean) 💥 - Показывать непрочитанные чаты в списке чатов первыми

  • munread_first (boolean) 💥 - Показывать непрочитанные сообщения первыми в мобильном приложении

  • can_add_to_team (boolean) 💥 - Могу ли я добавлять новых пользователей в эту команду

  • can_manage_sections (boolean) 💥 - Могу ли я управлять секциями в этой команде

  • can_manage_projects (boolean) 💥 - Могу ли управлять проектами в этой команде

  • can_manage_tags (boolean) 💥 - Могу ли я управлять тэгами в этой команде

  • can_manage_integrations (boolean) 💥 - Могу ли я управлять интеграциями в этой команде

  • can_manage_color_rules (boolean) 💥 - Могу ли я управлять правилами цветов в этой команде

  • can_create_group (boolean) 💥 - Могу ли я управлять групповыми чатами в этой команде

  • can_join_public_groups (boolean) 💥 - Могу ли я видеть/присоединяться к публичным группам в этой команде

  • can_join_public_tasks (boolean) 💥 - Могу ли я видеть/присоединяться к публичным задачам в этой команде

  • can_delete_any_message (boolean) 💥 - Устарело: используйте CanDeleteAnyMessage в объекте чата

  • custom_fields (ContactCustomFields) 💥 - Дополнительные поля контактов

Features¶

Информация о сервере. Только для чтения.

Поля:

  • host (string) - Текущий хост

  • build (string) - Версия компиляции сервера

  • desktop_version (string) - Версия приложения рабочего стола

  • front_version (string) - Версия веб приложения

  • app_title (string) - Титул приложения

  • landing_url (string) 💥 - Адрес изначальной страницы

  • app_schemes (array[string]) - Ссылки локальных приложений

  • userver (string) - аддресс сервера с статическими файлами

  • ios_app (string) - Ссылка в AppStore

  • android_app (string) - Ссылка в Google Play

  • theme (string) - Тема интерфейса по умолчанию

  • min_ios_version (string) - Минимальная версия приложения iOS требуемая на этом сервере. Используется для обратно несовместимых изменений

  • min_android_version (string) - Минимальная версия приложения Android требуемая на этом сервере. Используется для обратно несовместимых изменений

  • min_corp_ios_version (string) - Минимальная версия корпоративного приложения iOS требуемая на этом сервере. Используется для обратно несовместимых изменений

  • min_corp_android_version (string) - Минимальная версия корпоративного приложения Android требуемая на этом сервере. Используется для обратно несовместимых изменений

  • free_registration (boolean) - Свободная регистрация разрешена

  • max_upload_mb (number) - Максимальный размер загрузок пользователя

  • max_linked_messages (number) - Максимальное количество пересылаемых сообщений

  • max_message_uploads (number) - Максимальное количество загрузок в сообщении

  • max_username_part_length (number) - Максимальное количество букв для: фамилии, имени, отчества если существует

Поля:

  • uid (string) - Id тэга

  • name (string) - Имя тэга

Task¶

Task

Поля:

  • custom_color_index (number) 💥 - Настраиваемый цвет задачи

  • description (string) 💥 - Описание задачи

  • tags (array[string]) 💥 - Список тэгов задачи

  • section (string) 💥 - Id проекта / секции

  • observers (array[JID]) 💥 - Id подписчиков

  • items (array[string]) 💥 - Предметы задачи

  • assignee (JID) 💥 - Пользователь которому была дана задача

  • deadline (string) 💥 - Дедлайн задачи

  • public (boolean) 💥 - Является ли эта задача публичной

  • remind_at (string) 💥 - Послать уведомление в данное время

  • task_status (string) 💥 - Статус задачи

  • importance (number) 💥 - Важность задачи

  • urgency (number) 💥 - Срочность задачи

  • complexity (number) 💥 - Сложность задачи

  • spent_time (number) 💥 - Время затраченное на задачу

  • linked_messages (array[string]) 💥 - Присоединенные сообщения

  • uploads (array[string]) 💥 - Загрузки задачи

TaskTab¶

Вкладка задачи

  • title (string) - Титул вкладки

  • hide_empty (boolean) - Отключить ли вкладку если нет содержания

Team¶

Team

Поля:

  • uid (string) - Id команды

  • is_archive (boolean) 💥 - Команда была удалена

  • gentime (number) - Версия объекта

  • name (string) - Название команды

  • default_task_deadline (string) 💥 - Дедлайн задач по умолчанию

  • max_message_update_age (number) - Максимальное время обновления или удаления сообщений в секундах

  • icons (IconData) - Иконки команды

  • last_active (boolean) - Последняя активность пользователя была в этой команде

  • changeable_statuses (array[TeamStatus]) 💥 - Какие статусы я могу задавать другим участникам команды

  • bad_profile (boolean) 💥 - Мой профиль в этой команде не полноценный

  • need_confirmation (boolean) - Нудно подтверждение после приглашения в эту комманду

  • use_patronymic (boolean) 💥 - Использовать отчество в именах пользователей этой команды

  • user_fields (array[string]) - Порядок сортировки полей пользователей

  • display_family_name_first (boolean) 💥 - Фамилия должна быть первой в отображаемом имени

  • use_task_importance (boolean) 💥 - Использовать поле значимости в задачах

  • task_importance_min (number) 💥 - Минимальное значение важности. По умолчанию 1

  • task_importance_max (number) 💥 - Максимальное значение важности задач. По умолчанию 5

  • task_importance_rev (boolean) 💥 - Большая величина значит важнее. По умолчанию: меньшая величина значить важнее

  • use_task_urgency (boolean) 💥 - Использовать поле срочности в задачах

  • use_task_complexity (boolean) 💥 - Использовать поле сложности в задачах

  • use_task_spent_time (boolean) 💥 - Использовать затраченное время в задачах

  • uploads_size (number) 💥 - Всеобщий размер загрузок в байтах

  • uploads_size_limit (number) 💥 - Максимальный размер загрузок в байтах если существует

  • unread (tdproto-TeamUnread) 0️⃣ - Счетчики непрочитанных сообщений

  • me (Contact) - Мой профиль в этой комманде

  • contacts (array[Contact]) 💥 - Контакты команды. Только используется при создании комманды

  • single_group (JID) 💥 - Для команд с одной группой jid чата

  • theme (Theme) 💥 - Цветовая тема если существует

  • hide_archived_users (boolean) 💥 - Не показывать архивированных пользователей по умолчанию

  • pinned (boolean) 💥 - Закреплена ли команда

Upload¶

Загруженные медиа

Поля:

  • uid (string) - Id загрузки

  • created (string) - Во-сколько загружено

  • size (number) - Размер загрузки в байтах

  • duration (number) 💥 - Длительность (только для аудио или видео)

TaskFilter¶

Query для списка задач

  • assignee - * ?assignee=jid,jid

  • created_gte - * ?created_gte=<isodate>

  • created_lte - * ?created_lte=<isodate>

  • deadline_gte - * ?deadline_gte=<isodate>

  • deadline_lte - * ?deadline_lte=<isodate>

  • done_gte - * ?done_gte=<isodate>

  • done_lte - * ?done_lte=<isodate>

  • exclude_task_status - * ?exclude_task_status = new,done | new | any

  • gentime_gt - gentime great than group/chat

  • member - * ?member=jid,jid

  • num - * ?num=num1,num2,num3..

  • observer - * ?observer=jid,jid // TODO: rename to ?follower=

  • owner - * ?owner=jid,jid

  • public - * ?public=true|false

  • q - * ?q=

  • section - * ?section=[ uid,uid… | «-» ]

  • short - * ?short=true|false

  • sort - * ?sort = [ «created» | «-created» | «last_message» | «-last_message» | «deadline» | «-deadline» ]

  • tag - * ?tag=[ tag,tag,tag… | «-» ]

  • task_status - * ?task_status = new,done | new | any
