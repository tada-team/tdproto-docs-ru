Индекс JSON объектов¶

BotCommand¶

Информация о команде бота

    Поля:
    key (string) — Что должно быть вставлено в чат
    title (string) — Что должно быть видимо пользователем
    args (array[string]) — Опции команды если существует

ButtonColors¶

Цвета кнопок для приложения

    Поля:
    brand_static (RGBColor) — статический цвет
    brand_active (RGBColor) — активный цвет
    brand_disable (RGBColor) — отключенный цвет
    simple_static (RGBColor) — простой статический цвет
    simple_active (RGBColor) — простой активный цвет
    simple_disable (RGBColor) — простой отключенный цвет

CallDevice¶

Устройство участника звонка

    Поля:
    muted (boolean) — устройство заглушено
    useragent (string) — Описание устройства

CallOnliner¶

Участник звонка

    Поля:
    jid (JID) — Id контакта

    role (string) — Роль контакта

    muted (boolean) — Микрофон выключен. Вычисляется из состояния устройств
    devices (CallDevice) — Устройства участвующие, строго одно пока-что

Chat¶

Образ чата. (прямого, группового или задачного)

    Поля:
    jid (JID) — Id группы/задачи/контакта
    chat_type (ChatType) — Тип чата
    base_gentime (number) 💥 — Базовые поля (не относящиеся к каким либо участникам) версия
    gentime (number) — Поля чата относящиеся к какому либо конкретному участнику версия
    created (string) — Дата создания, iso datetime
    display_name (string) — Титул
    icons (IconData) — Данные иконки
    counters_enabled (boolean) 💥 — Непрочитанные сообщения будут учитываться в счетчике
    can_call (boolean) 💥 — Могу ли я звонить в этот чат
    can_send_message (boolean) 💥 — Могу ли я отправлять сообщения в этот чат
    cant_send_message_reason (string) 💥 — Почему я не могу отправлять сообщения в этот чат (если не могу)
    collapsed (boolean) 💥 — Сжатое описание. Используется только в задачах
    draft (string) 💥 — Черновик последнего сообщения, если существует
    draft_gentime (number) 💥 — Версия последнего черновика , если существует
    hidden (boolean) 💥 — Невидимый чат
    notifications_enabled (boolean) 💥 — Пуш уведомления включены
    num_importants (number) 💥 — Число важных сообщений
    num_unread (number) 💥 — Счетчик непрочитаных сообщений
    num_unread_notices (number) 💥 — Счетчик упоминаний (@)
    last_message (Message) 💥 — Объект последнего сообщения
    last_read_message_id (string) 💥 — Id последнего прочитанного сообщения, если существует
    section (string) 💥 — Id проекта / секции , если существует
    changeable_fields (array[string]) 💥 — Список редактируемых полей
    pinned (boolean) 💥 — Закреплен ли чат на верху
    pinned_sort_ordering (number) 💥 — Тип сортирования прикрепленного чата
    num_members (number) 💥 — Количество не архивированных участников
    can_delete (boolean) 💥 — Могу ли я удалить этот чат
    description (string) 💥 — Описание группы или задачи
    markup (MarkupEntity) 💥 — Markup объекты для описания. Экспериментально
    feed (boolean) 💥 — Существует в feed (главный экран)
    pinned_message (Message) 💥 — Прикрепленные сообщения этого чата
    color_index (number) 💥 — Индекс нестандартного цвета из таблицы цветов. Только в задачах
    num_items (number) 💥 — количество пунктов в чеклисте. Только в задачах
    num_checked_items (number) 💥 — Количество отмеченных пунктов в чеклисте. Только в задачах
    assignee (JID) 💥 — Id доверенного контакта. Только в задачах
    num (number) 💥 — Номер задачи в этой команде
    observers (JID) 💥 — Id подписчиков
    owner (JID) 💥 — Создатель задачи
    task_status (string) 💥 — Статус задачи. Возможно не стандартное
    title (string) 💥 — Титул задачи. Создается из номера и описания
    done (string) 💥 — Дата завершения задачи в iso формате, если существует
    done_reason (string) 💥 — Причина завершения задачи, если существует
    deadline (string) 💥 — Дедлайн задачи в iso формате, если существует
    deadline_expired (boolean) 💥 — Просрочен ли дедлайн
    links (MessageLinks) 💥 — Ссылки в описании
    tags (array[string]) 💥 — Список тэгов задачи, если существуют
    importance (number) 💥 — Важность задачи, если доступно в команде
    urgency (number) 💥 — Срочность задачи, если доступно в команде
    spent_time (number) 💥 — Время затраченное на задачу
    complexity (number) 💥 — Сложность задачи
    linked_messages (array[any]) 💥 — Используется для «Create task from messages…»
    uploads (Upload) 💥 — UID загрузок для запроса, объекты загрузок для ответа
    items (TaskItem) 💥 — Предметы чеклиста. Только в задачах
    parents (Subtask) 💥 — Родительские задачи
    tabs (TaskTabKey) 💥 — Имена закладок
    status (GroupStatus) 💥 — Мой статус в группе
    members (GroupMembership) 💥 — Участники группового чата
    can_add_member (boolean) 💥 — Могу ли я добавлять участников в этот групповой чат
    can_remove_member (boolean) 💥 — Могу ли я убирать участников из этого группового чата
    can_change_member_status (boolean) 💥 — Могу ли я менять статус участников этого группового чата
    can_change_settings (boolean) 💥 — арело: используете ``changeable fields``
    default_for_all (boolean) 💥 — Будут ли новые участники добавлены в этот чат при создании
    readonly_for_members (boolean) 💥 — Только для чтения для не администраторов (как каналы в Telegram но переключаемые)
    autocleanup_age (number) 💥 — Удалять сообщения через столько то секунд. Экспериментальная функция
    public (boolean) 💥 — Могут ли другие участники команды видеть этот чат/задачу
    can_join (boolean) 💥 — Могу ли я присоединиться к этому публичному чату/задаче
    can_delete_any_message (boolean) 💥 — Могу ли я удалять сообщения в этом чате
    can_set_important_any_message (boolean) 💥 — Могу ли я менять флаг важности сообщений в этом чате
    last_activity (string) 💥 — Дата последнего сообщения в этом чате если оно было удалено
    draft_num (number) 💥 — Устарело

Contact

    Поля:
    jid (JID) — Id контакта
    node (string) 💥 — uid узла для внешних пользователей

    is_archive (boolean) 💥 — Пользователь был удален
    botname (string) 💥 — Имя бота. Пустое для пользователей
    sections (array[string]) — Id секции
    can_send_message (boolean) 💥 — Могу ли я отправлять сообщения этому контакту
    cant_send_message_reason (string) 💥 — Почему я не могу отправлять сообщения в этот чат (если не могу)
    can_call (boolean) 💥 — Могу ли я звонить этому контакту
    can_create_task (boolean) 💥 — Могу ли я создавать задачи этому контакту

    alt_send (boolean) 💥 — Использовать Ctrl/Cmd + Enter вместо Enter
    asterisk_mention (boolean) 💥 — Использовать Ctrl/Cmd + Enter вместо Enter

    hide_pushes_content (boolean) 0️⃣ — Скрывать тело пушей

    quiet_time_start (string) 💥 — Начало тихого времени
    quiet_time_finish (string) 💥 — Окончание тихого времени
    focus_until (string) 💥 — Режим фокусировки включен до
    group_notifications_enabled (boolean) 💥 — Пуш уведомления для групповых чатов
    task_notifications_enabled (boolean) 💥 — Пуш уведомления для задач
    contact_short_view (boolean) 💥 — Короткий список контактов

    can_add_to_team (boolean) 💥 — Могу ли я добавлять новых пользователей в эту команду
